OCS WordCounter

The OCS WordCounter allows you to count the number of words that have more than a specified number of characters. The name comes from the essays assigned at Officer Candidate School, where candidates have to write an essay with an exact number of words, with only words over three letters counting towards the word count. In this tool, you can specify the minimum number of letters required to count as a word. You can see both the specialized word count as well as the total word count being updated as you type into the box.
